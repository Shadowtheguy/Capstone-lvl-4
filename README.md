# Creative Deck-nique, a MTG deckbuilder!
An express and react app that allows the user to create decklist and search up cards to add/remove from those lists

## Purpose
This app is for Magic the Gathering players who want to create and save decklists, and eventually even make custom cards and save them to custom lists

## MVP Description
Currently, the app can
- Produce only one decklist, but it is coded to allow others as they are added
- Search for any card either in the decklist, or a card using the search bar on the top
